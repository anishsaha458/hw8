Quiz App

This is a command-line quiz application matching the SPEC.md for the homework.

How to run:

1. Create a Python 3.8+ virtual environment (optional).
2. Install requirements: pip install -r requirements.txt
3. Run: python quiz.py

Data files:
- questions.json — human-editable question bank (already included)
- users.dat — binary pickled user data with bcrypt hashes
- history.dat — binary pickled per-user history
- feedback.dat — binary pickled per-user feedback
